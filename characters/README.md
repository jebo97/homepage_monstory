# 캐릭터 이미지 가이드

story.html(세계관 페이지)과 index.html(티저)에서 아래 파일명을 참조합니다.
파일이 없으면 자동으로 플레이스홀더(이모지+파일명)가 표시되니, 준비되는 대로 이 폴더에 넣으면 바로 교체됩니다.

## 필요한 이미지 (PNG · 투명 배경 권장)

| 파일명 | 쓰이는 곳 | 권장 내용 |
|---|---|---|
| `mosto-rito.png` | 메인 티저 + 세계관 히어로 | 모스토 🦖 + 리토 🐶 둘이 함께 있는 컷 ("우리 함께 성장해요" 일러스트) |
| `mosto.png` | 세계관 페이지 모스토 카드 | 모스토 단독 (코인 든 기본 포즈 추천) |
| `rito.png` | 세계관 페이지 리토 카드 | 리토 단독 (가방 멘 기본 포즈 추천) |

## 권장 사양
- 형식: PNG, 투명 배경
- 가로폭: 600~900px (히어로 합본 이미지는 1000px 이상 권장)
- 캐릭터 주변 여백은 최소화 (CSS에서 카드 안쪽 패딩으로 처리)

## 색상
- 모스토(Mosto): teal `#228b86`
- 리토(Rito): yellow `#FFE08A`
